【ターミナル】基本的なコマンド

ターミナル

ls

ディレクトリの内容を表示する

  • ls
  • ls -l 詳細情報を表示する
  • ls -a 隠しファイルも含めてすべて表示する
  • ls -t 更新日時順にソートして表示する

cd

ディレクトリの移動

  • cd to |toへ移動
  • cd ~ |homeディレクトリへ移動する
  • cd .. 一つ上のディレクトリへ移動する

pwd

現在のディレクトリを表示する

  • pwd

mkdir

ディレクトリの作成

  • mkdir etc 現在のディレクトリの中にetcディレクトリを作成する
  • mkdir -p etc/foo 中間ディレクトリも同時に作成する

rm

ファイルを削除する

  • rm sample.txt sample.txtを削除する
  • rm -r etc etcディレクトリを削除する

mv

ファイルを移動する/ファイル名を変更する

  • mv sample.txt etc sample.txtをetcディレクトリへ移動する
  • mv sample0.txt sample1.txt ファイル名を変更する

cp

ファイルをコピーする

  • cp sample1.txt sample2.txt

cat

ファイルを表示する/連結して表示する

  • cat foo.txt
  • cat foo.txt bar.txt

less

ファイルを閲覧する

  • j １行進む
  • k １行戻る
  • SPACE １ページ進む
  • b １ページ戻る
  • / 検索
  • n 次の検索結果
  • q 終了する
